# Langton's Ant

Langton's Ant to dwuwymiarowa uniwersalna maszyna Turinga z bardzo prostym zestawem reguł, a mimo to generująca złożone zachowania emergentne.

- **Stworzona przez**: Chris Langton (1986)
- **Typ**: Automat komórkowy
